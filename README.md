# Cyber-Bulling-from-Tweet

The Cyberbullying Detection from Tweets project is a machine learning-based system that automatically detects cyberbullying content from Twitter posts.
It helps social media platforms, researchers, and educators identify harmful content and take necessary actions to create a safer online environment.

**Features**

Detects cyberbullying content in tweets
Classifies tweets into categories like:
Hate Speech
Offensive Language
Non-offensive
Provides probability score for each category
Can be extended for real-time monitoring
User-friendly interface with Gradio / Streamlit

**Technologies Used**

Python
Scikit-learn
Pandas & NumPy (data preprocessing)
